# calendar_bot

[LINE登録](https://lin.ee/z1wVgXK)はこちら

### 概要
  ⇒ LINEアカウントで予定を追加し、今日の予定を聞くと返信が届き、予定がある前日21時にリマインドするBot。

### 機能一覧
  ・"説明"と入力すると、使い方をテキストで送信  
  ・日付指定で予定の追加（DBに追加）  
  ・"今日の予定"と入力すると、DBから予定を出力  
  ・予定がある前日の21時にリマインド（heroku schejuler）  

### 環境
  Ruby　2.7.1  
  Ruby_on_Rails　5.2.4.3  
  line-bot-api  
  whenever  
  mysql2  
  
### API
  line messaging api
